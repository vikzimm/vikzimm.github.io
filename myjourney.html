My tips for new coders

  1. First find the coding languages you like or that suits your learning needs
  2. Find multiple sources to learn. Coding is one of the best skills to learn online, and there is a lot of information available to you
  3. Stay motivated, find something you like to do to take a break from coding because it can be exhausting at times.
